uips-fixtures

Structured, minimal UiPath workflow examples — MCVEs (Minimum Complete Viable Examples), human- and LLM-readable, one pattern per repo, crowd-sourceable — with proven static analysis coverage and a controllable runtime test fixture.

Contribute

Each MCVE lives in its own repository. To contribute, fork _template-mcve, implement your pattern, and open a pull request. To request a new rule repo, open an issue in community.
